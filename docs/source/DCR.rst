==========================================
How to Download, Compile, and Run WRF-Fire
==========================================

In this section, the steps to compile WRF-Fire along with the required compilers and libraries are presented for different operating systems. HPC users who need to compile WRF-Fire using Singularity Container can use the related tutorial provided. To use this tutorial, you need to have a basic knowledge of Command Line and its mostly-used commands. In addition to compilation, detailed steps to run WRF-Fire for both Idealized and Real cases are provided in this section.
